# 포켓몬 도감 앱

## 프로젝트 소개
리액트를 사용한 포켓몬 도감 앱
> 개발기간: 2024.05.22 ~

## 기능
1. 포켓몬 목록 조회

## 추후 추가해야하는 기능
1. `더보기` 버튼 클릭하여 포켓몬 목록 더 조회하기
2. 포켓몬 카드 클릭 후 포켓몬 상세 페이지 조회하기
